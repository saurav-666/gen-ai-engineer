# Natural Language Processing

Use the NLP libraries to perform the following tasks:​

- [ ] Create a Word List​ from a text file
- [ ] Tokenize it​
- [ ] Use a Stemmer and a Lemmatizer​
- [ ] Filter out Stop Words​
- [ ] Identify the Parts of Speech​
- [ ] Perform Sentiment Analysis​
- [ ] Translate Phrase into a different language​
- [ ] Use NER to find the important Nouns in a phrase
- [ ] Build a Wordcloud

In the Resources folder, there is an example ```genai_essay.txt``` file created by Sidekick. As well as a Jupyter Notebook (```scrape.ipynb```)with a sample web scraping exercise to read the Smart Stories about Deloitte's Smart Factory. The results of the scraping are in the ```smart_stories.txt``` file.

Choose one of the two txt files to complete the exercises in this lab.
